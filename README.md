# Boilerplate Cpp

A boilerplate for C++ development using Cmake, Docker, CircleCI, and GoogleTests.

## Cloning the repository

```bash
$ git clone git@github.com:DouglasPimentel/boilerplate-cpp.git
```

## How to build and run

```bash
$ cd boilerplate-cpp
$ ./configure.sh
$ ./entrypoint.sh
```

## How to run tests

```bash
$ ./entrypoint.sh test
```
